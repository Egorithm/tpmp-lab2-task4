## Описание задачи
Описать структуру «Школьник» (содержащую подробный адрес и другие анкетные данные). Написать программу, которая реализует интерактивное меню для:
1. Создания текстового файла с произвольной информацией.
2. Просмотра содержимого исходного файла.
3. Чтения и обработки данных (поиск и выборка учеников 5-х классов).
4. Сохранения полученных результатов в новый текстовый файл и его просмотра.
5. Редактирования существующих записей прямо в файле.

Архитектура программы должна соответствовать модели КИС (Клиент-Интерфейс-Сервер).
